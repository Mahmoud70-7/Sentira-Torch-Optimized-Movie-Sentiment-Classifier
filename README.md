# Sentira-Torch-Optimized-Movie-Sentiment-Classifier
Deep Learning model using PyTorch to classify IMDb reviews. Features a custom NLP pipeline (Lemmatization &amp; Negation-aware Stopwords) with 87.6% accuracy. Includes an optimized 64-neuron MLP architecture and a Streamlit web UI for real-time sentiment analysis. (5000 TF-IDF features).
